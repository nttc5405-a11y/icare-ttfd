部署說明
  1. 開啟 Google Sheets，建立新試算表
  2. 上方選單 → 擴充功能 → Apps Script
  3. 將 icare-gas-backend.js 的內容貼上，儲存
  4. 修改第 8 行 TOKEN 為自訂密碼
  5. 右上角「部署」→「新增部署作業」→「網路應用程式」
  6. 執行身份：我；存取對象：所有人（含匿名）
  7. 複製部署後的 URL，貼入上方「GAS Web App URL」欄位
  8. 填入 TOKEN，按「儲存設定」並「測試連線」
